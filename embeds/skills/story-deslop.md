# 网文去AI味检测流程

## Gate 1: 词汇密度检测

检查以下AI高频词在每千字中出现的次数：
- "宛如/犹如/仿佛" > 2次/千字 → FAIL
- "不禁/不由自主" > 1次/千字 → FAIL
- "微微/缓缓/徐徐/悄然" > 3次/千字 → FAIL
- "竟然/居然" > 2次/千字 → FAIL
- "意味深长/若有所思" > 1次/千字 → FAIL

## Gate 2: 句式单一检测

- 连续3句以上使用"主语+说道"结构 → FAIL
- 连续3段以人名开头 → FAIL
- 段落平均长度偏差 < 20%（过于均匀） → FAIL

## Gate 3: 情感表达检测

- 使用"眼中闪过""瞳孔微缩""嘴角上扬"等模板化表情 > 3次/千字 → FAIL
- 心理活动全用"心中暗道""心想"引导 → FAIL
- 缺少具体的、个性化的感官描写 → FAIL

## Gate 4: 对话自然度检测

- 对话中频繁使用书面语 → FAIL
- 所有角色说话风格雷同 → FAIL
- 对话缺少语气词、省略、中断等自然特征 → FAIL

## Gate 5: 节奏检测

- 全文节奏均匀，无快慢变化 → FAIL
- 动作场景缺少短句和碎句 → FAIL
- 缺少段落间的留白和呼吸感 → FAIL

## Gate 6: 重复模式检测

- 相邻段落使用相同句式结构 → FAIL
- 同一场景中重复使用相同描写套路 → FAIL
- 过渡段使用"与此同时""正因如此"等AI连接词 → FAIL

## 真人写作基准

真人作者的文本特征：
- 词汇多样，很少重复使用同一修饰词
- 句式长短交错，节奏有变化
- 对话有口语感，不同角色语气不同
- 感官描写具体且有个人风格
- 段落长度不均匀，有呼吸感
- 心理活动融入动作和环境，不单独成段

## 检测报告模板

输出格式：
```
总字数: N
AI味指数: X/10 (1=完全自然, 10=重度AI味)

Gate 1 词汇密度: PASS/FAIL
  - 问题词汇: [列表]

Gate 2 句式单一: PASS/FAIL
  - 问题位置: [段落编号]

Gate 3 情感表达: PASS/FAIL
  - 模板化表情: [列表]

Gate 4 对话自然度: PASS/FAIL
  - 问题对话: [引用]

Gate 5 节奏: PASS/FAIL
  - 问题: [描述]

Gate 6 重复模式: PASS/FAIL
  - 重复位置: [描述]

修改建议:
1. [具体建议]
2. [具体建议]
```
