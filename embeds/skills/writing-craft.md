# 核心写作技法

## 章首钩子七式

开篇第一段必须抓住读者，以下七种方式任选：

1. **悬念钩**：抛出一个未解的问题或异常现象
2. **动作钩**：直接进入一个正在进行的动作场景
3. **对话钩**：以一句有冲击力的对话开场
4. **反差钩**：描述一个与预期完全相反的场景
5. **感官钩**：用一个强烈的感官印象（声音/气味/触感）开场
6. **时间钩**：点明一个关键时刻或倒计时
7. **情绪钩**：直接展现角色的极端情绪状态

## 章尾钩子十三式

每章结尾必须让读者想翻到下一页：

1. 新角色突然出现或旧角色以意想不到的方式回归
2. 主角发现一个关键秘密/线索
3. 突发危机/战斗即将开始
4. 角色做出重大决定但未揭示结果
5. 一句话反转之前的所有认知
6. 重要信息只透露一半
7. 角色面临两难选择
8. 环境突变（天气/场景/时间）
9. 收到一条改变局势的消息
10. 主角内心的恐惧/欲望被直接点明
11. 盟友突然表现出敌意
12. 一个看似完美的计划出现致命漏洞
13. 回扣第一章或前文的一个伏笔

## 爽点密度规则

- 每3000-5000字至少一个小爽点
- 小爽点：主角小胜、获得认可、展示能力、化解小危机
- 大爽点间隔不超过8000字
- 大爽点：重大突破、强敌落败、真相揭示、身份曝光
- 爽点前必须有足够的压抑/铺垫（至少等量篇幅）
- 爽后要有后果和影响，不要爽完就忘

## 节奏控制参考

- **快节奏**：短句为主（10字以内），段落2-3句，对话密集，动作连续
- **慢节奏**：长句可出现，段落5-8句，环境描写和心理活动充分
- **转换**：快→慢用一个"但是"或空行过渡；慢→快直接进入动作
- **战斗场景**：极短句，碎片化，感官聚焦，时间压缩
- **日常场景**：正常节奏，可穿插幽默，为后续蓄力
- **情感场景**：先慢后快或先快后慢，制造情绪波动

## 每章结构参考

```
开头（10%）: 钩子 + 承接上章
发展（60%）: 2-3个场景推进
高潮（20%）: 本章核心事件
结尾（10%）: 钩子 + 过渡
```
