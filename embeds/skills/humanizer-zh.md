# 中文去AI味规则

## 一、禁止模式（必须消除）

以下表达是AI生成文本的典型标志，必须替换：

1. "宛如" → 改用"好像""像是""跟……似的"
2. "犹如" → 改用"好像""仿佛"
3. "不禁" → 删除或改为"忍不住""一下子"
4. "眼中闪过一丝" → 改为具体的动作描写
5. "嘴角微微上扬" → 改为"笑了""咧嘴"
6. "心中暗道" → 直接写心理活动，不要加"心中暗道"
7. "竟然" → 减少使用频率，改用其他转折方式
8. "居然" → 同上
9. "不由自主" → 改为"忍不住""下意识地"
10. "若有所思" → 改为"想了想""琢磨着"
11. "意味深长" → 改为具体的语气或动作描写
12. "目光深邃" → 改为具体的眼神描写
13. "微微一笑" → 改为"笑了笑""咧了咧嘴"
14. "缓缓说道" → 改为"说""开口"
15. "淡淡地说道" → 改为"说"或直接省略
16. "沉声道" → 改为"压低声音说""低声说"
17. "嗤笑一声" → 改为"笑了""笑出声"
18. "瞳孔微缩" → 改为"一愣""吓了一跳"
19. "一道身影" → 改为具体的人物出现
20. "一股强大的气息" → 改为具体的感官描写
21. "在这一刻" → 删除或改为具体时间
22. "不得不说" → 删除
23. "说时迟那时快" → 删除，直接描写动作

## 三、口语化规则

1. 减少四字成语和书面词汇的使用密度
2. 对话中使用口语化表达："得了""行吧""算了""你说呢"
3. 避免连续使用排比句
4. 适当使用方言或习惯用语增加人物特色
5. 心理活动用短句、碎句，不要用长句

## 四、格式规范

1. 段落长度控制在3-6句，避免超长段落
2. 对话和动作交替出现，避免大段纯叙述
3. 避免连续3个以上段落都以人名开头
4. 场景转换要自然，不要用"与此同时"等连接词生硬切换
